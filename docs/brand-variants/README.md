# Brand mark variants (parked)

Preserved per the mark-redesign spec (docs/superpowers/specs/2026-07-17-mark-redesign-design.md §3):
options for a future "choose your mark" member-personalization feature and real choices for the
eventual mark-ratification founding ballot. Documentation only — nothing here ships in the bundle.

- `reference-classic.svg` — the pre-C2 mark (near-black star), lossless trace of the original.
- `spark-cutout.svg` (A) — 4-point navigation spark punched through the X. Sketch-grade geometry.
- `star-cutout.svg` (B) — 5-point star punched through the X. Sketch-grade geometry.
- `spark-glyph.svg` (D) — 4-point spark as a dark glyph on the X. Sketch-grade geometry.

Constraint recorded: PWAs cannot switch the installed home-screen icon dynamically — realistic
scope is in-app chrome/loading marks and possibly install-time choice.
